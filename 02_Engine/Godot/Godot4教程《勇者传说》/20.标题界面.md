# 20.标题界面

- 制作游戏标题界面（Title Screen）
- 掌握 Control 节点锚点系统和 Theme 主题系统
- 实现新游戏/读档/退出 三按钮功能

## 📖 内容笔记

### 1. 锚点系统（Anchor System）

Control 控件节点与 Sprite2D 不同，位置由**三组属性**决定：

| 属性组 | 含义 | 示例 |
|-------|------|------|
| **Anchor（锚点）** | 四条边相对父容器的百分比位置 | `left=0.5, right=0.5` → 水平居中 |
| **Anchor Offsets（偏移量）** | 相对锚点的像素偏移 | `left=-28` → 锚点左边 28px |
| **Grow Direction** | 内容超出时的扩展方向 | `both` → 左右均匀扩展 |

**两种拉伸模式**（项目设置→窗口）：

| 模式 | 行为 |
|------|------|
| **Keep** | 等比例缩放 + 黑边 |
| **Expand** | 等比例缩放 + 扩展画面（控件需要锚点才能居中） |

> 大多数情况下使用锚点预设菜单的 **16 个按钮**即可，不需要手动配三组属性。

### 2. 标题界面场景搭建

根节点 `Control`，保存于 `ui/title_screen.tscn`。

```
Control (TitleScreen)
├── TileMap (背景 — 从 forest 复制)
├── Decorations (前景装饰)
├── AnimatedSprite2D (玩家站立动画，6帧，10FPS)
├── Label (游戏名称"勇者传说")
│     → 锚点：顶部全服，horizontal = center
├── VBoxContainer (V)
│   ├── Button (NewGame — "新的故事")
│   ├── Button (LoadGame — "旧的回忆")
│   └── Button (ExitGame — "离开游戏")
│     → 锚点：右侧居中
└── AnimationPlayer (enter 入场动画)
```

#### 解决鼠标点击被拦截

选不到按钮 → 打开调试器"其他"选项卡查看谁在拦截 → 检查 `mouse → filter`：

| 节点 | 修改 |
|------|------|
| Game.ColorRect | `filter = Ignore` |
| Vignette.ColorRect | `filter = Ignore` |

### 3. 字体与主题系统

#### 为什么需要字体文件？
默认字体依赖操作系统 → 不同电脑显示可能不一致，没有中文字体则显示方块。

- 使用**开源免费商用字体**（如"得意黑"）
- 在 `Theme Overrides > Fonts` 中设置

#### 主题系统（Theme）查找顺序

```
节点本身的 theme_override
  → 节点的 theme 属性
    → 父节点的 theme 属性（逐级向上）
      → 项目设置 GUI → 自定义主题
        → 引擎默认主题
```

#### 创建全局主题

1. 根节点 → `Theme` → 新建 → 保存为 `main_theme.tres`
2. 展开主题编辑器：
   - **字体**：拖入字体文件
   - **Button → Styles**：`normal/hover/pressed/disabled` → 四个 `StyleBoxEmpty`（去掉默认背景）
   - **Button → Constants**：`outline_size = 6`（统一描边）
   - **Button → Colors**：`font_outline_color`、`font_pressed_color`（半透明）
   - **Focus → Styles**：`StyleBoxTexture` → 9 宫格缩放聚焦框

> [!tip] 9 宫格（Nine-patch）
> 调整四条虚线划分 9 个格子。上下 6 个格子拉伸时保持高度不变，左右 6 个格子保持宽度不变。用于自适应不同大小的控件。

### 4. 文本样式

| 效果 | 属性位置 |
|------|---------|
| 标题颜色 | Label → `Theme Overrides > Colors > Font Color` |
| 阴影颜色 | Label → `Theme Overrides > Colors > Font Shadow Color` |
| 阴影偏移 | Label → `Theme Overrides > Constants > Shadow Offset X/Y` |
| 阴影描边 | Label → `Theme Overrides > Constants > Shadow Outline Size`（=6）|

### 5. 按钮入场动画

AnimationPlayer → `enter` 动画（自动播放，0.5s）：
- V 节点的 `position`：从画面外右侧 → 最终位置
- 起始关键帧设置 `Easing` 向左拖 → 缓入减速效果

### 6. 键盘焦点（Focus）

```gdscript
# title_screen.gd — _ready()
new_game.grab_focus()                          # 默认聚焦"新的故事"

for button: Button in v.get_children():         # 鼠标移到哪，聚焦跟到哪
	button.mouse_entered.connect(button.grab_focus)
```

> ⚠️ `get_children()` 返回 `Array[Node]`，需要用 `button: Button` 做类型转换才能获得自动补全。

### 7. game.gd 新增功能

```gdscript
# game.gd — 新增变量
@onready var default_player_stats := player_stats.to_dict()    # 默认属性快照

# 新游戏
func new_game() -> void:
	change_scene("res://scenes/forest.tscn", {
		init = func():
			world_states = {}
			player_stats.from_dict(default_player_stats)
	})

# 返回标题
func back_to_title() -> void:
	change_scene("res://ui/title_screen.tscn")

# 存档检查
func has_save() -> bool:
	return FileAccess.file_exists(SAVE_PATH)
```

#### change_scene 兼容标题界面

标题界面不是 `World` → 添加类型判断：

```gdscript
# game.gd — change_scene() 关键修改
if tree.current_scene is World:              # 旧场景是 World 才序列化
	world_states[old_name] = tree.current_scene.to_dict()

# ... change_scene_to_file + await ...

if tree.current_scene is World:              # 新场景是 World 才恢复
	var new_name := tree.current_scene.scene_file_path.get_file().get_basename()
	if new_name in world_states:
		tree.current_scene.from_dict(world_states[new_name])
	# ...
```

> 为此 `world.gd` 需要在脚本开头定义 `class_name World`。

### 8. title_screen.gd 完整代码

```gdscript
extends Control

@onready var v: VBoxContainer = $V
@onready var new_game: Button = $V/NewGame
@onready var load_game: Button = $V/LoadGame

func _ready() -> void:
	load_game.disabled = not Game.has_save()
	new_game.grab_focus()
	for button: Button in v.get_children():
		button.mouse_entered.connect(button.grab_focus)

func _on_new_game_pressed() -> void:
	Game.new_game()

func _on_load_game_pressed() -> void:
	Game.load_game()

func _on_exit_game_pressed() -> void:
	get_tree().quit()
```
