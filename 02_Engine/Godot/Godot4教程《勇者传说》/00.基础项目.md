# 00.基础项目

## 📌 学习目标

## 📖 内容笔记
本教程使用 Godot 4.0.2 引擎，制作横版卷轴动作游戏。基础项目中创建了玩家角色（Player），使用 `CharacterBody2D` 节点实现基础的移动和跳跃。

### 角色基础逻辑
```gdscript
extends CharacterBody2D

const RUN_SPEED := 160.0          # 水平移动速度
const JUMP_VELOCITY := -320.0     # 跳跃初速度（向上为负）

var gravity := ProjectSettings.get("physics/2d/default_gravity") as float

@onready var animated_sprite_2d: AnimatedSprite2D = $AnimatedSprite2D

func _physics_process(delta: float) -> void:
    # 水平输入
    var direction := Input.get_axis("move_left","move_right")
    velocity.x = direction * RUN_SPEED
    # 重力
    velocity.y += gravity * delta

    # 跳跃检测
    if is_on_floor() and Input.is_action_just_pressed("jump"):
        velocity.y = JUMP_VELOCITY

    # 动画切换
    if is_on_floor():
        if is_zero_approx(direction):
            animated_sprite_2d.play("idle")
        else:
            animated_sprite_2d.play("run")
    else:
        animated_sprite_2d.play("jump")

    # 角色朝向
    if not is_zero_approx(direction):
        animated_sprite_2d.flip_h = direction < 0

    move_and_slide()
```

### 关键代码说明
- **`CharacterBody2D`**：Godot 4 中用于角色控制的物理体，内置 `move_and_slide()` 处理碰撞
- **`get_axis()`**：同时处理左右方向输入，返回 -1 / 0 / 1
- **`AnimatedSprite2D`**：管理角色动画（idle / run / jump），通过 `flip_h` 控制朝向

## 💡 关键要点
- 使用 `CharacterBody2D` + `AnimatedSprite2D` 构建角色
- `get_axis()` 处理左右移动输入
- `is_on_floor()` + `is_action_just_pressed("jump")` 实现跳跃
- `flip_h` 根据方向翻转角色朝向
