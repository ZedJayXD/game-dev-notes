# 14.交互

## 📌 学习目标
- 实现基础的交互

## 📖 内容笔记

### 1. Interactable 脚本

可交互对象的基础类，继承自 `Area2D`，通过玩家的 `CharacterBody2D` 进入/退出来检测交互。

#### interactable.gd

```gdscript
class_name Interactable
extends Area2D

signal interacted                    # 交互触发时发射

func _init() -> void:
	# 清理碰撞层和掩码
	collision_layer = 0
	collision_mask = 0
	# 只检测玩家（Layer 2）
	set_collision_mask_value(2, true)

func _ready() -> void:
	body_entered.connect(_on_body_entered)
	body_exited.connect(_on_body_exited)

func _on_body_entered(body: Player) -> void:
	body.register_interactable(self)

func _on_body_exited(body: Player) -> void:
	body.unregister_interactable(self)

func interact() -> void:
	print("Interacted with: ", name)
	interacted.emit()
```

> [!info] collision_layer / collision_mask 是位域
> 不是简单的整数，而是用位运算表示多个层的组合。设置时用 `set_collision_layer_value(layer, bool)` 和 `set_collision_mask_value(layer, bool)` 工具函数，不要直接赋值。

### 2. Player 交互系统

#### 变量定义

```gdscript
var interacting_with: Array[Interactable] = []    # 当前可交互对象列表

@onready var interaction_icon: AnimatedSprite2D = $InteractionIcon
```

#### 注册 / 注销函数

```gdscript
func register_interactable(interactable: Interactable) -> void:
	# 死亡状态下不接受新交互
	if state_machine.current_state == State.DYING:
		return
	if interactable in interacting_with:
		return
	interacting_with.append(interactable)

func unregister_interactable(interactable: Interactable) -> void:
	interacting_with.erase(interactable)
```

#### tick_physics 中更新图标可见性

```gdscript
# 在 tick_physics 中：
interaction_icon.visible = not interacting_with.is_empty()
```

#### 按键交互（_unhandled_input）

```gdscript
func _unhandled_input(event: InputEvent) -> void:
	if event.is_action_pressed("interact") and not interacting_with.is_empty():
		interacting_with.back().interact()  # 交互最近的可交互对象
```

### 3. 交互提示图标（AnimatedSprite2D）

使用 `AnimatedSprite2D`（不是 Sprite2D + AnimationPlayer），适合简单动画。

#### SpriteFrames 设置
- 新建 `SpriteFrames` 资源
- 默认已有一个 `default` 动画
- 点击表格按钮 → 选择素材图片（4 个键盘图标，16×16 像素）
- 按顺序点击四个 E 键图案 → 添加
- FPS 调整为 **6**（30 太鬼畜）

#### 场景结构

```
Player
├── ...
└── InteractionIcon (AnimatedSprite2D)
    └── SpriteFrames (新建资源)
```

> 默认 `visible = false`，进入交互区域时显示。

### 5. Bug 修复

#### 🐛 Bug 1：多个交互对象时，离开一个区域后无法交互

**原因**：`interacting_with` 是单个变量，离开任意交互区域都会清空。

**修复**：改为 `Array[Interactable]`，用 `register_interactable` / `unregister_interactable` 管理，按 E 时与数组最后一个元素交互（`back()`）。

#### 🐛 Bug 2：死亡后仍可交互

**原因**：死亡后 `body_entered` 仍会触发注册。

**修复**：
1. `register_interactable` 开头判断 `state_machine.current_state == State.DYING` → 直接 return
2. 进入死亡状态时清空数组：
```gdscript
# transition_state 中：
State.DYING:
	interacting_with.clear()
```
