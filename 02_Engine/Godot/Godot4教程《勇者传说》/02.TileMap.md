# 02.TileMap

## 📌 学习目标
- 掌握 Godot 4 中 TileMap / TileSet 的基本用法
- 学会用 TileSet 创建瓦片资源，用 TileMap 绘制关卡地图

## 📖 内容笔记
TileMap 是 Godot 4 中用于创建 2D 关卡布局的网格地图系统。核心由两个部分组成：

### TileSet（瓦片资源）
TileSet 是一个资源文件，定义了所有可用的瓦片图块。每个图块可以包含：
- **图像**：从瓦片图集中切割出的单个瓦片
- **碰撞形状**：为瓦片添加碰撞检测（如地面、墙壁）
- **导航多边形**：用于寻路系统
- **遮挡多边形**：用于光照系统

### TileMap（瓦片地图）
TileMap 使用 TileSet 中定义的瓦片，在网格上"绘制"出关卡地形。支持多层绘制，可以分别绘制地面层、装饰层、碰撞层等。

### 绘制流程
1. 创建或导入瓦片图集（TileSet Source）
2. 在 TileSet 编辑器中切割图块、设置碰撞形状
3. 将 TileMap 节点添加到场景
4. 在 TileMap 编辑器中用画笔/矩形/填充工具绘制地图
5. 整理 TileMap 的图层顺序（Z-index）

## 💡 关键要点
1. 新建 `TileMap` 节点（Godot 4.0.3 使用 `TileMap` 节点）
2. 在 Inspector 中新建 `TileSet` 资源
3. 进入 TileSet 编辑器，添加瓦片图集作为 Source
4. 在 TileSet 编辑器中为瓦片设置碰撞体（Physics Layer）
5. 回到 TileMap，用绘制工具在网格上画出关卡地形
6. 利用图层（Layers）分离地面和装饰
