# 🐛 问题记录

> Bug 追踪与解决方案沉淀

---

## 开放问题

```dataview
TABLE status as "状态", created as "创建日", tags as "标签"
FROM "09_Issues"
WHERE file.name != "🐛 问题导航" AND status != "resolved"
SORT created DESC
```

## 已解决

```dataview
TABLE created as "创建日", resolved as "解决日"
FROM "09_Issues"
WHERE status = "resolved"
SORT resolved DESC
```
